# MealLog - 食事記録アプリ

トレーニー向けの食事記録アプリです。増量期・減量期でPFC/カロリー目標を期間ごとに設定でき、日付が来たら自動で目標が切り替わります。

## 主な機能

- **今日の記録**：目標PFC/カロリーに対する残り・超過を表示。よく使う固定食品はワンタップで記録。音声入力（iPhone標準のディクテーション）にも対応
- **新しい食品を記録**：コンビニ購入品などを `品名,量,単位,kcal,P,F,C` の1行形式で記録。チェックを入れると固定食品としてマスタ登録もできる
- **固定食品マスタ**：g/個/杯で単位を指定。読みのゆれ（とりむね肉/鶏胸肉など）を別名として登録可能
- **増減量スケジュール**：期間ごとにPFC/カロリー目標を登録。今日が含まれる期間が自動で適用される
- **推移グラフ**：カロリー・体重の推移、直近7日のPFC平均達成率
- **体重記録**：毎朝の体重を記録し履歴・グラフで確認
- **週次まとめをコピー**：直近7日間の食事内容をテキストでコピーし、ChatGPTやClaudeに貼り付けて食事の改善相談ができる

## 使い方

### iPhoneへのインストール

1. Safari で `index.html` を開く（GitHub Pages等で公開したURL、またはローカルファイル）
2. 下部のシェアボタン → 「ホーム画面に追加」
3. ホーム画面からアプリを起動

### 初回セットアップ

1. 「設定」タブ → 「増減量スケジュール」で目標PFC/カロリーの期間を登録
2. 「設定」タブ → 「固定食品マスタ」で普段食べるものを登録（鶏むね肉、プロテインの銘柄ごとなど）
3. ホーム画面から記録を開始

## データ保存

- すべてのデータは端末のローカルストレージに保存されます
- クラウド同期・アカウント登録はありません（完全無料・オフライン動作）
- ブラウザのキャッシュを削除するとデータが失われるため注意してください
- 友人に渡す場合は同じファイル一式（またはURL）を共有してください。各自の端末で個別にデータが保存されます（データは共有されません）

## 技術仕様

- フロントエンド：HTML5 + CSS3 + JavaScript（ビルド不要）
- ストレージ：LocalStorage
- グラフ表示：Chart.js（CDN経由）
- PWA対応：オフラインで動作可能
