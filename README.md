# MealLog - 食事記録アプリ

トレーニー向けの食事記録アプリです。増量期・減量期でPFC/カロリー目標を期間ごとに設定でき、日付が来たら自動で目標が切り替わります。

## 主な機能

- **今日の記録**：目標PFC/カロリーに対する残り・超過を表示。よく使う固定食品はワンタップで記録。音声入力（iPhone標準のディクテーション）にも対応
- **日付の移動**：◀▶で前後の日に移動でき、記録し忘れた日を後から埋められる（その日の曜日に応じて目標・献立・クオータが自動で切り替わる）。未来には進めない
- **「前回と同じ」記録**：その食事を前に記録した内容をまるごとワンタップで記録
- **記録済みバッジ**：食事タブに印が付き、まだ入れていない食事がひと目でわかる
- **記録前の着地点プレビュー**：主菜を選び直すたびに「記録すると◯◯/1750kcal（残り◯◯）」が更新され、超過するかを選ぶ前に判断できる
- **取り消し**：記録直後にトーストの「取り消す」でまとめて取り消せる
- **食事スロット別メニュー**：朝食①・朝食②・昼食・間食・夕食のタブで切り替え（開いた時刻から自動選択）。タブごとにそのスロット専用のメニューだけが表示される
  - 朝食①②は固定献立。量を−/＋で調整して「この内容で記録する」のワンタップで完了（水曜は朝食②が自動で卵白＋ホエイに切り替わる）
  - 昼食・間食・夕食は「毎回食べるもの（白米・ブロッコリーなど）は固定＋主菜/主食はタップで選ぶ」構成。選んで量を調整し、まとめて記録
  - 夕食のレバーは任意枠（選んだときだけ記録）。水曜は主菜がサーモンで初期選択される
  - 量は前回記録した値が初期表示される（食事ごとに記憶するので、朝の白米200g／昼の白米150gは別々に覚える）
- **ワンタップ追加**：献立に無いものは「よく使う食品（★）」「最近使った食品」の折りたたみカード、固定食品一覧の「＋記録」、今日の記録の「＋」（同じ品をもう一度）からも記録できる
- **記録の編集**：今日の記録は食事ごとに小計付きでグループ表示。記録済みの量も行をタップして変更でき、カロリー・PFC・繊維・塩分が自動で再計算される
- **献立の編集**：設定 →「献立の編集」（またはホームの「献立を変更」）から、各食事の固定品・選択肢を追加/削除できる。任意枠や水曜の初期選択も設定可能。変更内容は保存され、いつでも初期状態に戻せる
- **PFCの修正**：メニュー各行の ✏️ から、その食品のカロリー・PFC・繊維・塩分をその場で修正できる（買う商品によって数値が変わるミックスナッツなどに対応）
- **周期化（カーボサイクル）**：トレ日/休み日の2種類、または曜日ごとに個別のPFC/カロリー目標を設定可能。水曜は「サーモンの日」として朝食②・間食の置き換えセットを優先表示
- **週次クオータ**：赤身肉 週2回・サーモン 週1回（水曜）・サバ缶 週2回・レバー 週1回（30g厳守、超過入力時は警告）の消化状況をホーム画面に表示。週の残りが2日を切ると未消化の枠を警告
- **食物繊維・食塩相当量**：PFCに加えて食物繊維（下限25g/日）・食塩相当量（上限7g/日）もトラッキング。推移グラフ・週次まとめにも出力
- **新しい食品を記録**：コンビニ購入品などを `品名,量,単位,kcal,P,F,C` の1行形式で記録。チェックを入れると固定食品としてマスタ登録もできる
- **固定食品マスタ**：g/個/杯で単位を指定。読みのゆれ（とりむね肉/鶏胸肉など）を別名として登録可能
- **増減量スケジュール**：期間ごとにPFC/カロリー目標を登録。今日が含まれる期間が自動で適用される。登録済みの期間は後から編集可能
- **固定食品の検索**：食品一覧を品名・読み（別名）で絞り込める
- **推移グラフ**：カロリー・体重の推移（体重は7日移動平均も表示）、直近7日のPFC平均達成率
- **体重記録**：毎朝の体重を記録し履歴・グラフで確認
- **週次まとめをコピー**：直近7日間の食事内容をテキストでコピーし、ChatGPTやClaudeに貼り付けて食事の改善相談ができる

## 使い方

### iPhoneへのインストール

1. Safari で `index.html` を開く（GitHub Pages等で公開したURL、またはローカルファイル）
2. 下部のシェアボタン → 「ホーム画面に追加」
3. ホーム画面からアプリを起動

### 初回セットアップ

1. 「設定」タブ → 「増減量スケジュール」で目標PFC/カロリーの期間を登録
2. 「設定」タブ → 「固定食品マスタ」で普段食べるものを登録（鶏むね肉、プロテインの銘柄ごとなど）
3. ホーム画面から記録を開始

## データ保存

- すべてのデータは端末のローカルストレージに保存されます
- クラウド同期・アカウント登録はありません（完全無料・オフライン動作）
- ブラウザのキャッシュを削除するとデータが失われるため注意してください
- 友人に渡す場合は同じファイル一式（またはURL）を共有してください。各自の端末で個別にデータが保存されます（データは共有されません）

## 技術仕様

- フロントエンド：HTML5 + CSS3 + JavaScript（ビルド不要）
- ストレージ：LocalStorage
- グラフ表示：Chart.js（CDN経由）
- PWA対応：オフラインで動作可能
